> Human-facing research branch focused on Uzbek mining, extractive industry,
> ESG materials, industrial memory, and the wider social-scientific and
> civilizational context inside the Terra ecosystem.

## Layer role

This repository is a human-facing research branch.

It should hold:

- mining-related reports and source materials;
- branch-level interpretation, comparison, and evidence;
- public-facing research surfaces around extraction, labor, and industry;
- release, citation, and audit surfaces for this branch.

It should not become:

- a broken mojibake shell;
- a fake DOI placeholder surface;
- a donor governance repository;
- an undifferentiated archive dump.

## Reading path

1. `README.md`
2. `CITATION.cff`
3. `STANDARDS.md`
4. `Publications/`
5. branch-specific reports, datasets, and comparative files

## Ecosystem position

- `FMP-CENTRAL-REPO` — central public hub
- `AIUZ-Terra-codex` — practice and ecosystem branch
- `AIUZ-terra-codex-FMP` — publication and FMP package layer
- `terra-legal` — donor governance and standards

## Operating rule

This repository must remain readable for humans first.

Its role is to expose an industrial and extractive branch of the ecosystem, not
to imitate the whole Terra structure inside one root file.
